HTML Quiz Answer Key

1. What does HTML stand for?

a. Hyper Text Markup Language

b. High Tech Modern Language

c. Hyperlink Text Management Language

d. Home Tool Markup Language

Correct Answer: a. Hyper Text Markup Language

2. Which HTML tag is used to define an internal style sheet?

a. <style>

b. <script>

c. <css>

d. <link>

Correct Answer: a. <style>

3. Which attribute is used to specify an alternate text for an image?

a. alt

b. src

c. title

d. href

Correct Answer: a. alt

4. What does CSS stand for?

a. Cascading Style Sheets

b. Creative Style Sheets

c. Computer Style Sheets

d. Colorful Style Sheets

Correct Answer: a. Cascading Style Sheets

5. Which HTML tag is used to create a hyperlink?

a. <a>

b. <link>

c. <href>

d. <nav>

Correct Answer: a. <a>

6. What is the default display property of a
element?

a. block

b. inline

c. flex

d. grid

Correct Answer: a. block

7. Which HTML tag is used to insert an image?

a. <img>

b. <picture>

c. <photo>

d. <src>

Correct Answer: a. <img>

8. Which input type is used for passwords?

a. password

b. text

c. secure

d. hidden

Correct Answer: a. password

9. Which HTML element is used to define a table row?

a. <tr>

b. <td>

c. <th>

d. <table>

Correct Answer: a. <tr>

10. What does the <br> tag do?

a. Adds a line break

b. Makes text bold

c. Creates a border

d. Starts a new paragraph

Correct Answer: a. Adds a line break

11. Which HTML element is used for the largest heading?

a. <h1>

b. <h6>

c. <head>

d. <header>

Correct Answer: a. <h1>

12. Which CSS property controls text size?

a. font-size

b. text-style

c. text-size

d. font-style

Correct Answer: a. font-size

13. Which CSS property sets the background color?

a. background-color

b. color

c. bgcolor

d. background

Correct Answer: a. background-color

14. Which HTML element is used to define a list item?

a. <li>

b. <ul>

c. <ol>

d. <list>

Correct Answer: a. <li>

15. Which CSS property controls the spacing between letters?

a. letter-spacing

b. word-spacing

c. line-height

d. text-align

Correct Answer: a. letter-spacing

16. Which HTML tag is used for a definition list?

a. <dl>

b. <ol>

c. <ul>

d. <list>

Correct Answer: a. <dl>

17. Which HTML attribute specifies the destination of a link?

a. href

b. src

c. alt

d. target

Correct Answer: a. href

18. What is the default value of the position property in CSS?

a. static

b. relative

c. absolute

d. fixed

Correct Answer: a. static

19. Which HTML element is used to create an ordered list?

a. <ol>

b. <ul>

c. <li>

d. <dl>

Correct Answer: a. <ol>

20. Which HTML tag is used to define a table?

a. <table>

b. <tr>

c. <td>

d. <th>

Correct Answer: a. <table>
